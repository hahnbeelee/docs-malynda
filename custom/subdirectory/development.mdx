## Installation and Preview

Install Mintlify globally:

<CodeGroup>

```bash npm
npm i -g mintlify
```


```bash yarn
yarn global add mintlify
```

</CodeGroup>

Start the local preview at `http://localhost:3000`:

```bash
mintlify dev
```

### Custom Ports

Use `--port` flag to specify a different port:

```bash
mintlify dev --port 3333
```

## Maintenance

Update CLI to match production version:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```


```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

Validate documentation links:

```bash
mintlify broken-links
```

## Troubleshooting

<AccordionGroup>
  <Accordion title='Error: Could not load the "sharp" module using the darwin-arm64 runtime'>
    1. Remove mintlify: `npm remove -g mintlify`
    2. Upgrade to Node v19\+
    3. Reinstall: `npm install -g mintlify`
  </Accordion>
  <Accordion title="Issue: Encountering an unknown error">
    Delete ~/.mintlify folder and run `mintlify dev` again.
  </Accordion>
</AccordionGroup>
